# OSUS Executive Sales Dashboard A modern, responsive sales dashboard for Odoo 17 with advanced analytics and KPI tracking. ## Features - **Real-time KPIs**: Track quotations, sales orders, invoiced amounts, and conversion rates - **Interactive Charts**: Monthly trends and distribution charts using Chart.js - **Responsive Design**: Mobile-friendly interface with professional styling - **Error Handling**: Robust fallback mechanisms and graceful degradation - **Field Compatibility**: Works with or without custom fields from other modules ## Requirements - Odoo 17.0+ - Modern web browser with JavaScript enabled - Internet connection for Chart.js CDN (fallback included) ## Installation 1. Copy the module to your Odoo addons directory 2. Update the app list in Odoo 3. Install the "OSUS Executive Sales Dashboard" module ## Usage Navigate to **Sales > Sales Dashboard > Executive Dashboard** to view the dashboard. The dashboard provides: - KPI cards with formatted values - Date range filtering - Sales type filtering (if available) - Monthly trend analysis - Revenue distribution charts - Recent sales activity table
