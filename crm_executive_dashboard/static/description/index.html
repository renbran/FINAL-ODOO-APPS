🚀 Installation Guide

  1. Copy the crm_executive_dashboard module to your Odoo addons directory
  2. Restart your Odoo server to discover the new module
  3. Go to Apps → Update Apps List
  4. Search for "CRM Executive Dashboard"
  5. Click Install and wait for installation to complete
  6. Navigate to CRM Executive → Executive Dashboard to access the module
  7. Configure dashboard settings and enjoy your new analytics!
