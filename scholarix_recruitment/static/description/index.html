# Scholarix Global - Recruitment Extension UAE-compliant offer letter system for Odoo 17 with professional branding. ## Features - Extended applicant tracking (Emirates ID, passport, nationality) - Automated salary calculations - Professional PDF offer letters - Email automation with branded templates - UAE Labour Law compliant ## Quick Start 1. Install: `Apps > Update Apps List > Search "Scholarix" > Install` 2. Configure: Upload company logo in Settings > Companies 3. Use: Recruitment > Applicants > Fill tabs > Generate Offer Letter ## Requirements - Odoo 17.0 - hr_recruitment module Navigate. Innovate. Transform.
